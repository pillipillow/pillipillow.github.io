Character Modelling Setup Tools (2019)

Character Modelling Setup tool is a quick start-up tool for modelling human characters from the ground up. Users can prepare front and side refs on a plane that will automatically set to origin position that will move above the grid, creates primitive shapes that are suitable for each body parts individually, or create all body parts in one go. These body parts uses the front reference as a guide to position the parts correspond to the reference picture (although not accurately). It also contains other shortcut tools that are mostly used during the modelling creation.

Role: Tools Programmer

For: Personal

Platform: 3DS MAX

Language: Maxscript
